# Set VitalIcon value to a Certain number

```sig
VitalIcons.setStatusBarValue(VitalIcon, 0)
```

## Parameters

* **Sprite**: the VitalIcon you want to set the value
* **number**: the value you want to set it to

NOTE: you can only set it as high as you made it to be
