# Postion StatusBar

this block positions a VitalIcon to a certain x and y on screen

```sig
VitalIcons.positionStatusBar(VitalIcon, 0, 0)
```
## Parameters
* **Sprite**: this is the StatusBar you want to Position
* **number**: the x position on screen
* **number**: the y position on screen
