# Change VitalIcon value by number

```sig
VitalIcons.changeStatusBar(VitalIcon, 0, VitalIcons.INPUT.FULL)
```
## Parameters

* **sprite**: the VitalIcon you want to change the value of
* **number**: the value you want to change the VitalIcon by
* **enum**: the type you want to change it by (make it go up/down by a full or a half)
