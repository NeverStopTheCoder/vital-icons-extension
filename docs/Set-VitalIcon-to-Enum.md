# Set VitalIcon to Full/Empty/Half

this block sets a VitalIcon to 

1.full (All the way filled)

2.empty (All the way drained)

3.half (just here because why not sets all the icons to a Half)

```sig
VitalIcons.setVitalIcon(VitalIcon, VitalIcons.VitalIconState.Full)
```
## Parameters

* **sprite**: the sprite you want to change
* **enum**: the type you want to set it to
