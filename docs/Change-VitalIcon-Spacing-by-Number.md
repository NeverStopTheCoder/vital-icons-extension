# Change the VitalIcon Spacing between icons by a Certain Number

```sig
VitalIcons.changeVitalIconSpacingBy(VitalIcon, 0)
```

## Parameters

* **sprite**: the VitalIcon that you want to change its spacing
* **number**: the value that you want to change its spacing by
