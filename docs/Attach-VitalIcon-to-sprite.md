# Atttach A VitalIcon to A Sprite

```sig
VitalIcons.attach(
mySprite,
VitalIcon
)
```
## Parameters

* **sprite**: the VitalIcon you are going to attach
* **sprite**: the sprite you are going to attach the VitalIcon to
* (Plus)
* **number**: the y offset value of the VitalIcon
* (Plus)
* **number**: the x offset value of the VitalIcon
