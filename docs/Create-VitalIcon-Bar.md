# Create Icons

a block that creates the Statusbar Icons with a certain Type/Custom image and set how many you want. 

```sig
let VitalIcon = VitalIcons.createStatusBar(
VitalIcons.VitalType.Hearts,
0
)
```
## Parameters

* **enum**: set the type to set a certain image of icons
* **Number**: set the number of icons in the Statusbar
* **(Plus)**
* **Image**: set the custom image for all filled/full versions of the icon
* **(Plus)**
* **Image**: set the custom image for all empty versions of the icon
* **(Plus)**
* **Image**: set the custom image for all half versions of the icon
